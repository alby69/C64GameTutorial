# Cartella Data

Questa cartella contiene risorse dati e metadati utilizzati dai tool di build e dal sito web.

## Contenuti

- **`wiki-graph.json`**: Contiene la definizione dei nodi (capitoli, concetti) e delle connessioni per la mappa interattiva del manuale. Viene elaborato da `tools/wiki-graph.py` per generare l'SVG navigabile.
