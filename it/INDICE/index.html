    Indice Analitico

    Elenco di registri, istruzioni, routine KERNAL e concetti, con riferimento al capitolo che li introduce.

    VIC-II — Registri Video ($D000–$D03F)

    Registro Nome Capitolo
    $D000–$D00F Posizione sprite X/Y (0-7) 5
    $D010 Sprite MSB X (bit 0-7) 6
    $D011 Controllo verticale / scroll fine Y 7, 24
    $D012 Raster line compare 7
    $D015 Sprite enable (bit 0-7) 5
    $D016 Controllo orizzontale / scroll fine X 17, 24
    $D017 Sprite double height (bit 0-7) 5
    $D018 Indirizzo char set / screen RAM 4
    $D019 Interrupt status register 7
    $D01A Interrupt enable register 7
    $D01B Sprite background priority (bit 0-7) 5
    $D01C Sprite multicolor mode (bit 0-7) 5, 6
    $D01D Sprite double width (bit 0-7) 5
    $D01E Sprite-sprite collision 10
    $D01F Sprite-background collision 10
    $D020 Border color 1
    $D021 Background color 0 1
    $D022–$D024 Background color 1-3 / sprite multicolor 5
    $D025–$D026 Sprite multicolor registers 5
    $D027–$D02E Sprite 0-7 color 5

    CIA — Registri di I/O

    CIA 1 ($DC00–$DC0F)

    Registro Nome Capitolo
    $DC00 Porta A — output joystick / colonna matrice 9
    $DC01 Porta B — input joystick / riga matrice 9
    $DC0D CIA1 interrupt control 7

    CIA 2 ($DD00–$DD0F)

    Registro Nome Capitolo
    $DD00 Porta seriale / bank VIC-II bit 0-1 21

    SID — Registri Audio ($D400–$D41C)

    Registro Nome Capitolo
    $D400–$D401 Voice 1 frequency 14
    $D402–$D403 Voice 1 pulse width 14
    $D404 Voice 1 control (gate/test/ring/ sync/rect/tri/saw/noise) 14
    $D405–$D406 Voice 1 ADSR 14
    $D407–$D40C Voice 2 (stessa struttura) 14
    $D40E–$D413 Voice 3 (stessa struttura) 14
    $D414 Filter cutoff frequency low 14
    $D415 Filter cutoff frequency high / resonance 14
    $D416 Filter mode (lp/bp/hp) 14
    $D417 Volume + filter enable 14
    $D418 Volume (0-15) 14

    KERNAL — Routine di Sistema ($FF81–$FFF3)

    Indirizzo Nome Descrizione Capitolo
    $FF81 IOBASE Indirizzo base I/O —
    $FF84 SCINIT Inizializza schermo —
    $FF87 PLOT Posiziona cursore —
    $FF90 RDTIM Legge clock di sistema —
    $FF99 UDTIM Aggiorna clock —
    $FF9C RDTIM2 Legge time (alternativa) —
    $FFA2 STOP Controlla tasto STOP —
    $FFA5 RSSTAT Stato RS-232 —
    $FFA8 GETIN Legge carattere da tastiera —
    $FFAB CLRCH Chiude canali I/O —
    $FFAE CHKIN Apre canale input —
    $FFB1 CHKOUT Apre canale output —
    $FFB4 CLRCN Pulisce canale —
    $FFB7 CHRIN Legge byte da canale —
    $FFBA SETLFS Imposta logical file/device/command 21
    $FFBD SETNAM Imposta nome file 21
    $FFC0 OPEN Apre file —
    $FFC3 CLOSE Chiude file —
    $FFC6 CHKIN2 Versione alternativa CHKIN —
    $FFC9 CHKOUT2 Versione alternativa CHKOUT —
    $FFCC CLRCH2 Versione alternativa CLRCH —
    $FFCF BASIN Get byte (versione BASIC) —
    $FFD2 BSOUT Output carattere (CHROUT) 1
    $FFD5 LOAD Carica file da disco 21, 23
    $FFD8 SAVE Salva file su disco 23
    $FFDB SETTIM Imposta clock —
    $FFDE MEMTOP Legge top memoria —
    $FFE1 MEMBOT Legge bottom memoria —
    $FFE7 SETBNK Imposta bank —
    $FFEA SETMSG Controllo messaggi —
    $FFED SETTMO Timeout device —
    $FFF0 IECIN Input IEC —
    $FFF3 IECOUT Output IEC —
    $EA31 Interrupt return con Acknowledge Fine IRQ (ack + RTI indiretto) 7

    6502/6510 — Istruzioni

    Trasferimento dati

    Istruzione Descrizione Capitolo
    LDA #n Carica immediato in A 1
    LDA addr Carica da indirizzo in A 1
    LDX #n / LDX addr Carica immediato/da indirizzo in X 2
    LDY #n / LDY addr Carica immediato/da indirizzo in Y 2
    STA addr Salva A in memoria 1
    STX addr Salva X in memoria 2
    STY addr Salva Y in memoria 2
    TAX Trasferisci A → X 2
    TAY Trasferisci A → Y 2
    TXA Trasferisci X → A 2
    TYA Trasferisci Y → A 2

    Aritmetica

    Istruzione Descrizione Capitolo
    ADC #n Addizione con carry 2
    SBC #n Sottrazione con carry 2
    INC addr Incrementa memoria 2
    DEC addr Decrementa memoria 2
    INX / INY Incrementa X / Y 2
    DEX / DEY Decrementa X / Y 2

    Logiche e bit

    Istruzione Descrizione Capitolo
    AND #n AND logico 2
    ORA #n OR logico 2
    EOR #n XOR logico 2
    LSR A Logical shift right 2
    ASL A Arithmetic shift left 2
    ROL A Rotate left 3
    ROR A Rotate right 3
    BIT addr Bit test 2

    Salti e salti condizionali

    Istruzione Descrizione Capitolo
    JMP addr Salto incondizionato 2
    JSR addr Salta a subroutine 2
    RTS Ritorno da subroutine 2
    BEQ addr Salta se zero (==) 2
    BNE addr Salta se non zero (!=) 2
    BCC addr Salta se carry clear (<) 2
    BCS addr Salta se carry set (>=) 2
    BMI addr Salta se negativo 2
    BPL addr Salta se positivo 2
    BVC addr Salta se overflow clear 2
    BVS addr Salta se overflow set 2

    Comparazione

    Istruzione Descrizione Capitolo
    CMP #n Compara A 2
    CPX #n Compara X 2
    CPY #n Compara Y 2

    Stack

    Istruzione Descrizione Capitolo
    PHA Push A sullo stack 3
    PLA Pull A dallo stack 3
    PHP Push status sullo stack 3
    PLP Pull status dallo stack 3
    TSX Stack pointer → X 3
    TXS X → stack pointer 3

    Interrupt

    Istruzione Descrizione Capitolo
    SEI Set interrupt disable 7
    CLI Clear interrupt disable 7
    RTI Return from interrupt 7

    Altre

    Istruzione Descrizione Capitolo
    NOP No operation 2
    BRK Breakpoint forzato (software) 22

    Mappa di Memoria

    Indirizzo Dimensione Descrizione Capitolo
    $0000–$00FF 256 byte Zero page 3
    $0100–$01FF 256 byte Stack hardware 3
    $0200–$03FF 512 byte KERNAL/BASIC work area —
    $0400–$07FF 1024 byte Screen RAM (default) 4
    $0800–$9FFF 38912 byte BASIC / memoria programma 1
    $0801 — Inizio programma BASIC 21
    $C000 — Indirizzo standard assemblaggio 1
    $A000–$BFFF 8192 byte BASIC ROM (o RAM) —
    $C000–$CFFF 4096 byte Area programma alternativa 6
    $D000–$D3FF 1024 byte VIC-II registers 4
    $D400–$D41C 29 byte SID (audio) 14
    $D800–$DBFF 1024 byte Color RAM 4
    $DC00–$DCFF 256 byte CIA 1 9
    $DD00–$DDFF 256 byte CIA 2 21
    $FF80–$FFFF 128 byte KERNAL ROM (jump table) 21

    Concetti per Capitolo

    Capitolo Concetti principali
    1 LDA/STA, $D020/$D021, assemblaggio a $C000, TMPx, BSOUT $FFD2
    2 Istruzioni base, loop (DEX/BNE), delay (NOP), ADC/SBC, AND/ORA/EOR, CMP/CPX/CPY, INC/DEC, flag
    3 Zero page, X-indexed, Y-indexed, indirect, stack (PHA/PLA/TSX/TXS), ROL/ROR
    4 Screen RAM ($0400), Color RAM ($D800), $D018, PETSCII, caratteri personalizzati
    5 Sprite enable $D015, posizioni $D000–$D00F, pointer $07F8, dati sprite, colori, multicolor
    6 Movimento sprite, MSB $D010, animazione, multicolor sprite, double size
    7 IRQ setup (SEI/CLI), $D012, $D019, $D01A, $0314/$0315, raster bar, JMP $EA31
    8 Frame counter, 50 Hz loop, sincronizzazione verticale
    9 $DC01, direzioni, edge detection, debouncing
    10 Bounding box, $D01E/$D01F, collision detection software
    11 Pool/spawn proiettili, cooldown timer, active/inactive
    12 Onde nemiche, pattern AI, difficolta progressiva
    13 Punteggio BCD/esadecimale, state machine (MENU/PLAY/GAMEOVER)
    14 SID, waveform (tri/saw/pulse/noise), ADSR, filter, volume $D418
    15 Audio engine, coda SFX, music engine, priority
    16 Multiplexing software (8+ sprite), raster split per sprite
    17 Raster split, scroll $D016, parallax due layer
    18 Boss multi-fase, pattern attack, weak point
    19 3-layer (INPUT/LOGIC/RENDER), scheduler, entity system
    20 Arcade OS concepts, interrupt chaining, self-modifying code, 3-layer kernel
    21 SETNAM $FFBD, SETLFS $FFBA, LOAD $FFD5, caricatore personalizzato
    22 VICE monitor, breakpoint b, watchpoint w, disasm d, step x/z, raster debug
    23 Schermata titolo animata, SAVE $FFD8, LOAD high score, game over screen
    24 Scroll fine $D016/$D011, scroll grossolano, raster split scroll, parallax scroll, vertical scroll

    Modificatori di Indirizzamento

    Sintassi Nome Capitolo
    #n Immediato 1
    $C000 Assoluto 3
    $C000,X Assoluto indicizzato X 3
    $C000,Y Assoluto indicizzato Y 3
    $C0 Zero page 3
    $C0,X Zero page indicizzato X 3
    ($C000) Indiretto (JMP) 3
    ($C0),Y Zero page indiretto indicizzato Y 3
    ($C0,X) Zero page indiretto pre-indicizzato X 3
    label Indirizzamento relativo (branch) 2

    VICE — Comandi del Monitor

    Comando Azione Capitolo
    r Mostra registri CPU 22
    m $addr Mostra memoria 22
    d $addr Disassembla 22
    b $addr Breakpoint su esecuzione 22
    w $addr Watchpoint su accesso 22
    wl $addr Watchpoint su lettura 22
    ws $addr Watchpoint su scrittura 22
    g Continua esecuzione 22
    x Step over 22
    z Step into 22
    bc Cancella breakpoint 22
    bl Lista breakpoint 22
    a $addr Assembla in memoria 22
    l "file" $start $end Salva memoria su file 22
