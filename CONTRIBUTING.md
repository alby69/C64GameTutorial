# Linee Guida per Contribuire

Grazie per l'interesse nel contribuire a **C64 Game Tutorial**! Questo progetto mira a fornire il miglior materiale possibile per chi vuole imparare a programmare giochi arcade sul Commodore 64.

## Come puoi aiutare

- **Segnalazione di errori:** Se trovi refusi nel testo, errori nei diagrammi o bug nelle soluzioni assembly, apri una Issue.
- **Traduzioni:** Aiutaci a mantenere e migliorare la traduzione inglese in `en/`.
- **Esercizi:** Se hai idee per nuovi esercizi o varianti interessanti per i capitoli esistenti.
- **Tooling:** Miglioramenti agli script in `tools/` o al Makefile.

## Processo di Pull Request

1.  Fai un fork del repository.
2.  Crea un branch per la tua modifica (`git checkout -b feature/miglioramento-testo`).
3.  Esegui `make validate` per assicurarti che non ci siano errori di consistenza.
4.  Invia la Pull Request descrivendo chiaramente i cambiamenti.

## Stile e Convenzioni

- **Markdown:** Segui la struttura esistente. Usa i tag `### Esercizio` correttamente perché vengono usati dallo script di validazione.
- **Assembly:** Usa i commenti per spiegare le parti complesse. Mantieni le label coerenti con lo stile del tutorial.
